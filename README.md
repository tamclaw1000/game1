# Game1

A small native SwiftUI 2048 game with separate iOS and macOS app targets that share the same game model and UI.

## Run

Open `Game1.xcodeproj` in Xcode, then select either:

- `Game1 iOS` for iPhone or iPad simulators/devices
- `Game1 macOS` for Mac

The iOS app uses swipe gestures. The macOS app supports swipe gestures and arrow keys.
